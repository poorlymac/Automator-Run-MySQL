# Automator-Run-MySQL
An Automator Action in Swift that uses native MySQL libraries to run queries. Please keep in mind this is alpha quality code that I pulled together over a day, that said it works well for me üëç

# Requirements
Install the MySQL client & OpenSSL using `brew install mysql openssl`

# Current Supported Databases
* [MySQL](https://www.mysql.com) e.g. `mysql://user:password@hostname/database`
* [PostgreSQL](https://www.postgresql.org) e.g. `postgresql://user:password@localhost/postgres`
